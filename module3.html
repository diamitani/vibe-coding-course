Hosted App Builders

Lovable
Hosted Builder
Lovable is an AI full-stack app builder. Describe your project in natural language and Lovable generates a complete application with frontend, backend, database, and authentication. It uses React + Supabase under the hood and gives you a live preview as it builds.
Full-Stack Supabase One-Click Deploy
v0.dev
Hosted Builder ¬∑ by Vercel
v0 generates production-ready UI components from text descriptions. Built by Vercel, it outputs React + Tailwind code that's ready to drop into any Next.js project. Ideal for front-end design and component generation.
UI Components React Tailwind
Bolt.new
Hosted Builder ¬∑ by StackBlitz
Bolt.new gives you an instant full-stack development environment in the browser. Powered by WebContainers, it runs Node.js entirely in your browser ‚Äî no server needed. Describe your app and watch it get built in real time with instant previews.
Full-Stack WebContainers No Server
Replit
AI IDE ¬∑ Cloud-Based
Replit combines a cloud IDE with an AI agent that can build entire applications from a description. It handles the development environment, hosting, and deployment ‚Äî making it the most "all-in-one" option available. Great for both learning and production.
All-in-One Cloud IDE Built-in Hosting

AI Agents & CLI Tools

Claude Code
CLI Agent ¬∑ by Anthropic
Claude Code is Anthropic's agentic coding tool that lives in your terminal. It reads your entire codebase, understands project structure, and makes multi-file changes directly. Powerful for complex projects where you need deep context awareness.
Terminal Multi-File Deep Context
Codex
CLI Agent ¬∑ by OpenAI
OpenAI's Codex is a cloud-based coding agent that runs tasks in a sandboxed environment. It can read your repository, execute code, run tests, and submit pull requests. Designed for integration into professional development workflows.
Sandboxed GitHub Integration Pull Requests
OpenCode
CLI Agent ¬∑ Open Source
OpenCode is an open-source terminal-based AI coding assistant. It's provider-agnostic, meaning you can plug in any LLM (OpenAI, Anthropic, local models). Great for developers who want full control and transparency over their tooling.
Open Source Any LLM Transparent
Antigravity
Agentic IDE ¬∑ by Google DeepMind
Antigravity is Google DeepMind's advanced agentic coding assistant. It operates inside an IDE with full access to terminal, browser, and file system. It plans, executes, and verifies changes ‚Äî making it one of the most powerful agentic tools available.
Agentic Browser + Terminal Full IDE

How to Choose the Right Tool

There's no single "best" tool ‚Äî it depends on your goals, skill level, and what you're building:

  • Just getting started? ‚Üí Start with Lovable or Bolt.new. They have the gentlest learning curve.
  • Want beautiful UI? ‚Üí Use v0.dev for component design, then bring the code into your project.
  • Need everything in one place? ‚Üí Replit covers IDE, hosting, and deployment.
  • Working on an existing codebase? ‚Üí Claude Code or Antigravity can read your whole project and make intelligent changes.
  • Want maximum control? ‚Üí OpenCode is fully open source and works with any model.
üí° Pro Tip

Don't commit to just one tool. Expert vibe coders often chain tools together ‚Äî use v0 to generate a component, Lovable to scaffold the app, and Claude Code to refine the details.
